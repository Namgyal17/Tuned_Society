# Garage Logos

Store garage logo images in this directory.

## Naming Convention
Name the image files using the garage's **slug** (URL-friendly name).
The application will automatically try to load `[slug].png` for each garage.

**Format:** PNG (Preferred)
**Dimensions:** Square (e.g., 200x200px) recommended.

### Examples:
- **Garage Name:** Wheelerz Hub -> **Slug:** `wheelerz-hub` -> **File:** `wheelerz-hub.png`
- **Garage Name:** Auto Meister -> **Slug:** `auto-meister` -> **File:** `auto-meister.png`

## How it works
1. Drop the image file here.
2. The site will automatically display it in the Consultation Modal.
3. If no image is found, the site will fallback to displaying the garage initials.
